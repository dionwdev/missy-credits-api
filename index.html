Missy Elliott API

Each album as an object containing title, wiki cover image source, release date, studio recorded, length, label, producers, and tracks.

Albums tracks key contains an array of objects for each track. Track objects contain track number, title, array of features, array of writers, array of producers, and length.

Endpoints: https://missy-credits-api.herokuapp.com/:album name with no spaces or punctuation
